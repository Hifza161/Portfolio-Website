Project Overview

The Complaint Management System (AI-CMS) is a web-based system designed to manage, track, and resolve university complaints efficiently using AI-based categorization and automated workflows.

The system supports students, staff, and administrators with role-based dashboards and real-time complaint tracking.

Key Features

  • Role-based authentication (Student, Staff, Admin)
  • AI-based complaint categorization & urgency detection
  • Complaint tracking with unique ID system
  • Workflow-based assignment & resolution system
  • Feedback and complaint reopening feature
  • Admin dashboard with analytics & reports
