Travel App

A smart travel planning app designed in Figma with features like itinerary planning, budget tracking, and safety tools — focused on a clean, intuitive user experience.

UX/UI Design
Figma
Completed

Project Overview

This project was designed to address the pain points travelers face when planning trips — scattered information, budget chaos, and safety uncertainty. The app brings everything under one clean, intuitive interface.

The focus was on user research-informed design, clean visual hierarchy, and making sure the core workflows (itinerary + budget) were as frictionless as possible.

Key Features

  • Smart itinerary builder with drag-and-drop day planning
  • Budget tracker with category-wise spending overview
  • Safety alerts and emergency contact quick-access
  • Offline-friendly trip card layouts
  • Clean onboarding flow for first-time users

Design Process

Started with user research and identified three core pain points: disorganized planning, overspending, and feeling unsafe in unfamiliar places. Wireframes were created for all major screens, then iterated into high-fidelity Figma prototypes with a consistent design system.

The color palette uses calm teals and warm oranges to feel both trustworthy and adventurous — reflecting the spirit of travel.
