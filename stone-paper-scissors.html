Project Overview

Rock Paper Scissors is a modern browser-based game designed with clean visuals, responsive layouts, and smooth gameplay interactions.

The project focuses on JavaScript event handling, random game logic, score tracking, and interactive UI animations to create an engaging user experience.

Main Features

  • Interactive player vs computer gameplay
  • Real-time score tracking system
  • Randomized computer move generation
  • Smooth animations and hover effects
  • Responsive design for all screen sizes

Key Challenges

One of the biggest challenges was implementing accurate game logic while creating smooth and interactive user feedback during gameplay.

  • Generating random computer choices dynamically
  • Managing score updates in real-time
  • Handling all possible game outcomes correctly
  • Creating responsive layouts and animations

What I Learned

This project improved my understanding of JavaScript conditions, event listeners, randomization logic, and DOM manipulation.

It also helped me learn how to build more interactive frontend applications with smoother user experiences.
