Project Overview

This lab project explored how intelligent agents navigate problem spaces using informed search strategies. The goal was to implement and compare algorithms that find optimal or near-optimal paths from a start state to a goal state on a grid.

Both A* and Greedy Best-First Search (GBFS) were implemented from scratch in Python and tested on several grid configurations with varying obstacles and goal positions.

Algorithms Implemented

A* Search
Uses f(n) = g(n) + h(n) — combines actual cost from start with heuristic estimate to goal. Guarantees the optimal path.
Greedy BFS (GBFS)
Uses only h(n) — the heuristic estimate to the goal. Faster but not always optimal. Good for time-constrained scenarios.

  • Implemented Manhattan distance as the heuristic function
  • Explored how obstacle density affects algorithm performance
