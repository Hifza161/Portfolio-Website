EventoHub

A full-stack Android application for end-to-end event management — from creation and discovery to user coordination and real-time notifications.

Android
Kotlin
Firebase
Full-Stack

Overview

EventoHub solves a real problem — organizing and attending events is fragmented across multiple apps. EventoHub consolidates event creation, discovery, RSVP management, and notifications into one clean Android experience.

Built with Kotlin and powered by Firebase, it delivers real-time data sync so organizers and attendees always see the latest event updates instantly.

Key Features

  • Event creation with title, description, date, location, and capacity limit
  • User authentication and role management (Organizer / Attendee)
  • Real-time push notifications for event updates and reminders
  • RSVP system with live attendee count tracking
  • Firebase Firestore for real-time database sync
  • Clean Material Design UI with smooth navigation

Tech Stack

Kotlin Android Studio Firebase Firestore Firebase Auth FCM Notifications Material Design

What I Learned

This project taught me how to architect a real-time Android app from scratch — handling authentication flows, Firestore data modelling, and Firebase Cloud Messaging for push notifications. It strengthened my understanding of MVVM patterns and asynchronous data handling in Kotlin.
